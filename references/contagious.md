# Contagious (STEPPS) Framework
**Core Concept:** Why things catch on via word-of-mouth.

### The STEPPS Elements
- **Social Currency:** Does sharing this make the user look smart or "in the know"?
- **Triggers:** Is there an environmental cue? (e.g., "Coffee" triggers "KitKat").
- **Emotion:** High-arousal emotions (Awe, Anger, Excitement) drive sharing.
- **Public:** "Built to show, built to grow." Is the usage of the product visible?
- **Practical Value:** Is it useful? (e.g., A "Top 10" list or a quick hack).
- **Stories:** Wrap the product in a narrative that is hard to tell without the brand.

### Verification Checklist
- Does this provide the sharer with "Social Currency"?
- Is there an everyday "Trigger" that will remind them of this next Tuesday?
