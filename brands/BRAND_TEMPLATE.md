# How to create a new brand profile

**Option A (recommended):** Run `/setup` in your agent — it will interview you and save the profile automatically.

**Option B (manual):** Copy the template and fill it in yourself:
```bash
cp assets/BRAND_TEMPLATE.md brands/my-brand.md
# Edit brands/my-brand.md, then validate:
bash scripts/validate-brand.sh brands/my-brand.md
```
